# Pixel Guide (Android)

Download and import game guides on your Android device

Pixel Guide is a clean, offline-first game guide reader for Android.  
It was inspired by the Pocket Codex app on iOS, but built specifically for Android devices.

Pocket Codex can be found here: https://www.pocketcodex.app/

If you enjoy browsing classic GameFAQs guides, maps, and charts, or you have your own guides you want to keep neatly organized, Pixel Guide is designed for you.

---

## What is Pixel Guide?

Pixel Guide allows you to:

- Browse game guides from GameFAQs
- Download guides directly to your device
- Read guides offline in a clean, distraction-free interface
- Import your own guides in TXT, HTML, or PDF format

The app is optimized for readability and works especially well on handheld gaming devices and tablets.

---

## Feature Highlights

### Reading Experience
- Clean, distraction-free guide reader
- Fast in-guide text search for TXT and HTML guides
- Table of Contents support for HTML guides
- Embedded images in HTML guides for a complete offline experience
- Grid or list layout on the home screen

### Guide Support
- Download and view TXT, HTML, and PDF guides
- Import your own local guides
- Full offline access once guides are downloaded
- Support for maps and charts

### Library Management
- Browse guides per game
- Manually add custom games
- Organize all guides locally on your device

### Language and Translation
- Multi-language app interface support:
  - English
  - Spanish
  - Portuguese
  - German
  - French
  - Italian
  - Turkish
  - Polish
- On-device translation for TXT guides powered by Google ML Kit
- Translations run fully offline after downloading language models
- Supports 59 languages

# Pixel Guide (Android)

Browse, save, import, and read game guides on your Android device.

Pixel Guide is a clean guide browser and saved-guide reader for Android. It connects to supported guide sources to find and save guides, then renders saved and imported guide content locally on your device, including when no network connection is available.

## Important Note About Offline Use

Pixel Guide is not a fully offline app.

An internet connection is required to browse online guide sources, find new guides, download guides, save web pages, download manuals, use Pixie AI, and open external shortcut URLs.

Once guide content has been saved or imported, Pixel Guide can render that content locally on your device, including without a network connection. TXT guide translation also works without a connection after the required Google ML Kit language models have been downloaded.

---

## What is Pixel Guide?

Pixel Guide is an Android app for collecting and reading game reference material in one place.

It lets you browse GameFAQs and Neoseeker guides, save complete HTML walkthroughs directly on your device, download manuals, import your own files, track RetroAchievements progress, and keep saved content available for local rendering.

If you enjoy classic walkthroughs, GameFAQs guides, Neoseeker TXT and HTML guides, maps, charts, PDF manuals, achievement hunting, or your own personal game notes, Pixel Guide is designed for you.

The app is optimized for readability and works especially well on handheld gaming devices, tablets, and Android phones.

It was inspired by the Pocket Codex app on iOS, but built specifically for Android devices.

Pocket Codex can be found here: https://www.pocketcodex.app/

## Feature Highlights

### Web Guides

- Browse and download guides directly from GameFAQs
- Support for GameFAQs TXT and HTML walkthrough downloads
- Support for Neoseeker TXT guides
- Support for Neoseeker HTML walkthrough downloads, including multi-page guide saving
- Save complete single-file HTML guides directly on your Android device without a desktop, transfers, or imports
- Built-in ad blocking while browsing supported web guides
- Remember guide pages and entrypoints for GameFAQs and Neoseeker
- Back and forward navigation inside the HTML reader
- Add shortcut URLs for external guides, wikis, and custom resources

### Reading Experience

- Clean, distraction-free guide reader
- Smart Text Formatting for classic TXT guides, with intelligent screen-width reflow
- Optional justified TXT text for a more book-like layout
- Table of Contents support for TXT guides
- Fast in-guide text search for TXT and HTML guides
- Table of Contents support for HTML guides
- Optional bottom Table of Contents navigation bar for HTML guides
- HTML guide images fit the screen, with original-size viewing on tap
- PDF fit-to-width reading
- Resume reading from where you last stopped
- Per-guide font size and font selection
- Theme-aware TXT and HTML guide display
- Controller navigation for readers, including page turning and scrolling
- Fine-tuned controller scrolling for smoother gamepad and handheld navigation
- Multi-window support for launching two Pixel Guide instances side by side
- Saved guide content can render locally without network access

### TXT Guide Navigation

Pixel Guide can detect and generate navigation sections for many classic TXT walkthroughs, making it easier to jump between chapters, areas, bosses, items, secrets, and other guide sections.

TXT guide Table of Contents detection supports many classic guide styles, including GameFAQs and Neoseeker-style text guides, but old TXT guides are highly varied and not always consistently formatted. Some TOCs may not be detected perfectly yet. Please report broken or missing TOCs so they can be improved.

- Enable controller support from Settings > Other Settings > Enable Controller Support

---

## Privacy

Pixel Guide is privacy-first:

- No telemetry
- No tracking
- No background analytics

Your guide library stays on your device. Built-in ad blocking helps keep supported web guide pages cleaner while saving and reading guides.

Pixie AI is optional and only works after you provide your own Gemini API key. Requests made through Pixie are sent to Google's Gemini service according to your configured API usage.
